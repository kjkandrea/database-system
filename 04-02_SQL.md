# 데이터 삽입, 수정, 삭제

## INSERT 문

* 테이블에 새로운 레코드를 삽입
* 모든 속성 또는 부분 속성(컬럼)에 대한 속성값을 삽입

```sql
INSERT INTO [테이블 이름]
    VALUES(값1, 값2, ...)
```

```sql
INSERT INTO [테이블 이름(컬럼1, 컬럼2, ...)]
    VALUES(값1, 값2, ...)
```

## UPDATE 문

조건을 만족하는 레코드의 특정 컬럼값을 수정

```sql
UPDATE [테이블 이름]
    SET 컬럼=값
    [WHERE 조건]
```

## DELETE 문

조건에 일치하는 레코드 집합을 테이블에서 삭제할 때 사용하는 언어

```
DELETE FROM [테이블 이름]
    [WHERE 조건]
```

## SAFE UPDATES 모드

의도하지 않은 데이터 변경/삭제 방지를 위해 WHERE절이 없는 UPDATE/DELETE문은 실행 하지 않음

## SELECT 문

한 개 이상의 테이블에서 주어진 조건에 만족하는 레코드를 출력

* SELECT
* FROM
* ON/WHERE
* GROUP BY
* HAVING
* ORDER BY
